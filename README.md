## Tech stack

- Next.js 15 (app router) + React 19
- Tailwind CSS
- Shadcn UI
- Tanstack Query

## Installation

### Prerequisite

- Node.js 18 or higher
- pnpm

### Steps

- Clone this project
- Install the packages: `pnpm i`
- Run the development server: `pnpm dev`
- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
